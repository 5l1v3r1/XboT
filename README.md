# XboT
Simple Facebook messenger bot with php

Facebook bot อย่างง่าย เขียนด้วย PHP ครับ

การติดตั้ง
1.ต้องมี App บน Facebook ผูกไว้กับ page โดย เลือกเป็น Messenger
2.ต้องมี Site HTTPS/SSL เป็น Webhook
3.แก้ค่า 'xxxx' ในไฟล์
